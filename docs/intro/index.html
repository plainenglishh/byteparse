Datatypes

Introduction​

Types​

u8​

Description: An unsigned 8 bit integer.
Size: 1 Byte
Encoding: Native

u16​

Description: An unsigned 16 bit integer.
Size: 2 Bytes
Encoding: Native

u32​

Description: An unsigned 32 bit integer.
Size: 4 Bytes
Encoding: Native

i8​

Description: An signed 8 bit integer.
Size: 1 Byte
Encoding: Native

i16​

Description: An signed 16 bit integer.
Size: 2 Bytes
Encoding: Native

i32​

Description: An signed 32 bit integer.
Size: 4 Bytes
Encoding: Native

f32​

Description: An 32 IEEE 754 floating point.
Size: 4 Bytes
Encoding: Native

f64​

Description: An 32 IEEE 754 floating point.
Size: 8 Bytes
Encoding: Native

varint​

Description: A ProtoBuf Base 128 Variable-width integer.
Size: 1..10 bytes
Encoding: See https://protobuf.dev/programming-guides/encoding/#varints

bool​

Description: A boolean value (1/0)
Size: 1 Byte
Encoding: Encoded as either a 0 or 1 u8

null​

Description: A null byte (0)
Size: 1 Byte
Encoding: Encoded as a 0 u8

string​

Description: A raw string
Size: Any
Encoding: String written directly with no further metadata

lstring​

Description: A length-prefixed string
Size: 0..10 + 0..2^64 Bytes
Encoding: String written directly with a varint prefixed, denoting length

zstring​

Description: A null terminated string
Size: Any
Encoding: String written directly followed by a null byte

array​

Description: An array of serialisable values
Size: Any
Encoding: Array written directly with no further metadata

larray​

Description: A length-prefixed array of serialisable values
Size: 0..10 + 0..2^64 Bytes
Encoding: Array written directly with a varint prefixed, denoting length

zarray​

Description: A null terminated array of serialisable values
Size: Any
Encoding: String written directly followed by a null byte

buffer​

Description: A raw buffer
Size: Any
Encoding: Buffer written directly with no further metadata

lbuffer​

Description: A length-prefixed buffer
Size: 0..10 + 0..2^64 Bytes
Encoding: Buffer written directly with a varint prefixed, denoting length

zbuffer​

Description: A null terminated buffer
Size: Any
Encoding: Buffer written directly followed by a null byte

vector​

Description: A 3 or 4-width vector
Size: 12 or 16 Bytes, depending on vector size
Encoding: 3 or 4 f32s

table​

Description: A serialisable luau table
Size: Any
Encoding:

has_array = u1
has_dict = u1

if has_array then
    array_length = varint
    ...array_value = any -- (repeated for 1..array_length)
end

if has_dict then
    dict_length = varint
    ... {
        key = any
        value = any
    } -- (repeated for 1..dict_length)
end

any​

Description: Any serialisable luau value
Size: Any
Encoding: A Type ID followed by a the values
