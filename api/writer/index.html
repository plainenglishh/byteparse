write_zstring​

</>
writer.write_zstring(
value: string,--

The string to write.

size: number?--

The maximum size of the string.

) → ()

Writes a null terminated (C Style) string.

WARNING

This function doesn't check whether the passed value is null-safe, and will write the entire string regardless of embedded null bytes.

Consider wrapping the passed value with byteparse.trim_zstring if you can't be sure there are no embedded null bytes.
